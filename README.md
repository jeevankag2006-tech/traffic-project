# 🚦 Smart Traffic Management System using AI

## 📌 Overview

This project is an **AI-based Traffic Management System** that uses computer vision and deep learning to analyze traffic from video input and optimize traffic flow.

The system detects vehicles using YOLO and processes traffic density in real-time to simulate intelligent traffic control.

---

## 🎯 Features

* 🚗 Vehicle Detection using YOLOv8
* 📊 Traffic Density Analysis
* 🎥 Video Upload & Processing
* 🌐 Web Interface using Flask
* ⚡ Real-time Simulation

---

## 🛠️ Tech Stack

* Python
* Flask
* OpenCV
* YOLOv8 (Ultralytics)
* HTML, CSS

## ⚙️ Installation

### 1️⃣ Clone the repository

```
git clone https://github.com/jeevankag2006-tech/traffic-project.git
cd traffic-project
```

### 2️⃣ Install dependencies

```
pip install ultralytics opencv-python flask
```

---

## ▶️ Run the Project

```
python app.py
```

Open browser:

```
http://127.0.0.1:5000
```
